# Codebook — AI-Readiness on Business Websites (2026)

Versioned definitions for the released dataset. Every measured field is derived only from public
HTTP responses (homepage HTML, `/robots.txt`, `/llms.txt`). Raw HTML is **not** redistributed; only
parsed fields are released.

## Dataset files (`data/`)

| File | Rows | Role in paper |
|------|------|---------------|
| `crawl_confirmatory_766.jsonl` | 766 | Primary. All reported figures come from this crawl (reachable n=594). |
| `crawl_replication_487.jsonl` | 487 | Replication sample. |
| `crawl_pilot_149.jsonl` | 149 | Pilot / first replication sample. |

Each line is one site (one JSON object). Join to `sample_meta.csv` on `domain` for `vertical`,
`city`, and `stratum`.

## Record fields (one JSON object per site)

| Field | Type | Definition |
|-------|------|------------|
| `domain` | string | Registered domain from the OpenStreetMap `website` tag. |
| `timestamp_utc` | ISO 8601 | When the site was crawled. |
| `home_status` | int / null | HTTP status of the homepage fetch (after `https`→`http` fallback). `null` = network error. |
| `reachable` | bool | `true` iff a homepage returned HTTP 200 with non-empty body. All adoption stats are computed over `reachable == true`. |
| `platform` | string | Heuristic CMS/builder fingerprint (headers + HTML markers). `"unknown"` when no marker matched. |
| `schema_types` | list[str] | Sorted unique `@type` values extracted from all JSON-LD `application/ld+json` blocks on the homepage, walked recursively. |
| `has_identity_schema` | bool | `schema_types` intersects the IDENTITY set (see below). |
| `has_trust_schema` | bool | `schema_types` contains `Review` or `AggregateRating`. |
| `has_qa_schema` | bool | `schema_types` contains `FAQPage` or `QAPage`. |
| `has_person_schema` | bool | `schema_types` contains `Person`. |
| `robots` | object / null | Per-bot status map (see AI-crawler list). `null` when no valid `robots.txt` was served. |
| `ai_bots_blocked` | list[str] | Subset of AI bots with `Disallow: /` (full block). Length 0 = blocks none. |
| `llms_txt` | object | `/llms.txt` result (see llms.txt rules). |

## Schema type sets (v1)

- **IDENTITY:** `Organization`, `LocalBusiness`, `AccountingService`, `ProfessionalService`,
  `Attorney`, `Dentist`, `MedicalBusiness`, `HomeAndConstructionBusiness`, `Store`, `Corporation`,
  `LegalService`, `RealEstateAgent`, `HealthAndBeautyBusiness`, `DaySpa`, `BeautySalon`, `Plumber`.
- **TRUST (review):** `Review`, `AggregateRating`.
- **QA (FAQ):** `FAQPage`, `QAPage`.
- **PERSON:** `Person`.

## AI-crawler list (v1, 14 user-agents)

`GPTBot`, `ChatGPT-User`, `OAI-SearchBot`, `Google-Extended`, `ClaudeBot`, `anthropic-ai`,
`Claude-Web`, `PerplexityBot`, `Perplexity-User`, `CCBot`, `Bytespider`, `Amazonbot`,
`Applebot-Extended`, `Meta-ExternalAgent`.

Per-bot `robots` status values:
- `not_mentioned` — no matching `User-agent` group in robots.txt.
- `mentioned` — a group targets the bot but does not `Disallow: /`.
- `blocked_all` — a group targeting the bot contains `Disallow: /`. Counted in `ai_bots_blocked`.

## llms.txt validity + provenance rules (v1)

`/llms.txt` counts as **present** only if it returns HTTP 200 AND:
1. the first 1024 bytes contain no HTML/redirect/error markers (`<html`, `<!doctype`, `<head`,
   `<body`, `301 moved`, `302 found`, `not found`, `<?xml`, `page not found`);
2. body length ≥ 40 bytes after trimming;
3. body has markdown structure (contains `#` or a `](http` link).

`provenance` (checked in the first 500 bytes, first match wins): `RankMath` ("rank math"),
`Yoast` ("yoast"), `generated_other` ("generated by"), else `manual`. "Tool-generated" in the paper
= any value other than `manual`; this is a **lower bound** (unsigned generated files read as `manual`).

## AI-Readiness index (pre-registered, 0–10)

`+2` allows all AI crawlers (`len(ai_bots_blocked) == 0`); `+2` valid llms.txt; `+2` identity schema;
`+2` review schema; `+1` FAQ schema; `+1` Person schema. Operational benchmark, not a validated
latent construct — see paper §3.

## Statistics

Proportions: Wilson score 95% CI (correctly bounded at 0%/100%). Index mean: bootstrap 95% CI
(≥2000 resamples, fixed seed 12345). Between-vertical claims rest on non-overlapping 95% intervals.
Reproduce with `python3 analyze.py data/crawl_confirmatory_766.jsonl sample_meta.csv`.
